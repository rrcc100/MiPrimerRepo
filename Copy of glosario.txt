git init: Inicia un nuevo repositorio Git en un directorio existente o vacío.
git clone: Clona un repositorio Git existente desde una URL remota a tu directorio local.
git add: Agrega archivos al área de preparación (staging) para el próximo commit.
git commit: Registra los cambios en el repositorio. Es necesario después de utilizar git add.
git status: Muestra el estado actual del árbol de trabajo y el área de preparación.
git log: Muestra el historial de commits del repositorio.
git branch: Lista, crea o elimina ramas.
git checkout: Cambia entre ramas o restaura archivos desde el área de preparación.
git merge: Fusiona una rama con otra.
git pull: Obtiene los cambios desde un repositorio remoto y los fusiona con tu rama local.
git push: Envía los commits locales a un repositorio remoto.
